# 03 · Rules

## The 8 rules

| # | Rule |
|---|------|
| 01 | **Solo or team** — no cap on size; everyone must be registered and on the floor |
| 02 | **Pick one track** — Virality, Revenue, or AI as Agency. Chosen at registration; that rubric is your primary score. Wins outside your track = bonus points only |
| 03 | **Use Hermes, one of two ways** — (a) coding partner that built your product (keep session receipts), or (b) base harness your end users interact with (show ≥1 capability doing real work). Both allowed. **No Hermes, no score** |
| 04 | **Build on-site** — 8-hour sprint on the floor. Mentors watch builds all day. No remote teammates, no outside code |
| 05 | **Fresh builds only** — no pre-existing products, not your company's product. Ideas/sketches/scaffolding fine |
| 06 | **Submit in the window** — opens after sprint ends. Late = not considered, no exceptions |
| 07 | **Numbers get verified** — signups checked live in DB, traffic cross-checked, customers called, emails bounce-tested. Spoofed number = parameter zeroed |
| 08 | **Judges' decision is final** — no appeals |

## Rule 05 — valid starting point (the test: built today, on this floor?)

### ✅ Qualifies
- Starting from zero today
- Starter template/boilerplate extended substantially during the sprint
- An idea sketched/wireframed before but never built or deployed
- Standard scaffolding: Next.js, Vite, FastAPI, create-anything starters
- BaaS: Supabase, Firebase, Convex, Clerk — wiring infra is not pre-building
- LLM SDKs, APIs, AI coding assistants doing the heavy lifting (that's the point)

### ❌ Does not qualify
- Finished product with cosmetic changes made today
- Fork of your own/someone's project where today's work is superficial
- Your company's existing product in any wrapper
- Remote contributors or off-floor code during the sprint
- Something already demoed elsewhere in current form
- A build using Hermes in neither qualifying way

### Borderline?
**Flag it in your submission.** Honest flags almost always survive the mentor check. Hiding the origin = auto-disqualification if caught.
